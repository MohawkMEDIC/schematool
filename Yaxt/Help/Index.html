Overview

The MARC-HI Schema Tool is a lightweight yet powerful tool for working with XML Schemas. It offers many features including:

  • An XML Schema Viewer which outlines the structure of data
  • An XML Schema Differencing tool which illustrates changes in schema structures
  • An XPath building tool (in the Schema Viewer). This tool allows developers to quickly build Xpaths based on schema structures
  • An XML Schema Copy tool which copies all dependent schemas.

This help file is divided into these sections:

  • Getting Started - A quick guide to Schema Tool's interface
  • Viewer - A guide to the schema viewer
  • Differencing - A guide to interpreting the output of the differencing tool
  • Schema Copy - A guide for using the schema copy tool
