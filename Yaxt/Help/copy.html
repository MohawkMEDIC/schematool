Xml Schema Copy

The schema copy tool will copy selected and all dependent schemas to a specified directory. The schema copy tool window is outlined in the following image:

  1. Source Schema(s) - Click the ellipsis to select schema(s) that you would like copied (holding SHIFT while click files will select multiples)
  2. Destination Directory - Select the directory that you would like your schema file(s) copied to (note: an empty directory is preferred as it will less likely result in overwritten files)
  3. Progress / Status - The file currently being copied/processed will be shown here
  4. Copy - Self explainatory... Starts the copy process.
